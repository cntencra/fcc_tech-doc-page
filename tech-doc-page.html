What is git

Git is a tool for keeping track of code changes. Using git you can more easily create different versions of your code and work on these different versions simultaneoulsy with other developers. Git allows you to simply revert back to previous versions of your code or work on improving parts of your code in branches without changing the main branch. This can allow for bug fixes that can be fully tested before being rolled out to users.

Basic git workflow

Below is a basic workflow for using git on a daily basis

  • Write code
  • Check status - git status
  • Stage changes - git add .(everything) or filename (parts of a file can also be added)
  • Commit changes - git commit
  • Push changes - git push (push changes up to a repository)
Github

Github is an online repository for storing your code. the git push comand pushes changes up to an online repository. There are many different online repositories Github being one of these.
